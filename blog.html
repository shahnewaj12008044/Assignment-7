1. Difference between map, filter, forEach, find

map is a method of array which returns a new array with some condintions.It can change the elements of the array.

forEach is similar to map but forEach does not return a new array.By using it we can modify an array but cant store the modified array in a variable whereas we can do that by using map.

Filter is the method of array which is actually used to find the specific elements of an array which satisfies the condition.It technically dont change the original array.

Find is similar to filter but find only returns the first value that satisfies condition whereas filter returns an array with all the elements that satisfies the condition.

2. Difference between Local storage, Session storage, cookies.

Local Storage stores data of users in string data type with no expiration date in the broweser.Its maximum capacity is 5 Mb.

Session storage stores data of users in string data type but only for the session means it will last until the browser opened when someone close the tab the data gets deleted but after reloading the page it remains same.The maximum capacity of session storage is 5-10 MB.

Cookies are small files of information that a web server generates and sends to a web browser for improving user experience.It can be accessed by document.cookie
